Projeto Final: Engenharia de Qualidade e Automação Front-End

PintheDogs é um catálogo digital interativo para entusiastas de cães que, inspirado no estilo do Pinterest, utiliza APIs para exibir e organizar fotos de diversas raças em uma galeria visualmente atraente.

Tecnologias
Este projeto foi desenvolvido com as seguintes tecnologias:

HTML

CSS

JAVASCRIPT

THE DOG API


Projeto

O PinTheDog é uma aplicação que oferece uma experiência personalizada de descoberta de raças através de uma interface temática (Front-end) e um sistema funcional de busca em tempo real com salvamento de favoritos via localStorage (Back-end/JS). Além disso, a aplicação foi pensada com foco em boas práticas de engenharia de qualidade, incluindo organização modular do código, reutilização de componentes e testes manuais para validação das principais funcionalidades.

A interface foi projetada para ser responsiva, garantindo uma navegação fluida em diferentes dispositivos, como desktops, tablets e smartphones. A experiência do usuário também foi otimizada com carregamento dinâmico de imagens e feedback visual durante as interações.


Funcionalidades Principais

Busca de raças em tempo real
Exibição de imagens dinâmicas consumidas da API
Sistema de favoritos com persistência em localStorage
Layout responsivo inspirado no Pinterest
Atualização dinâmica do conteúdo sem recarregar a página


Instalação e Execução

Para rodar o projeto localmente, siga os passos abaixo:

Clone o repositório:

git clone <url-do-repositorio>

Acesse a pasta do projeto:

cd nome-do-projeto

Instale as dependências:

npm install

Inicie a aplicação:

npm start

Abra o site no navegador por meio do link gerado ou utilizando uma extensão como o Live Server.
